PHPonCLI

A single abstract class to create a Command Line Interface (CLI) with PHP.

Made for rapidly building command line apps - it includes:

  • Full featured opts/args parser
  • Simply add commands by adding a new method to the class
  • Spelling suggestions
  • Output colored/styled messages
  • Command auto-completion and glob support
  • Help based on method documentation

Overview

Requirements

  • PHP >= 5.3

Download

Just bundle PHPonCLI.php with your app.

Via GIT: git clone --recursive git://github.com/phponcli/phponcli.git ~/git/phponcli You can replace ~/git/phponcli with whatever you want.

Usage

Terminology

  • Command = top level command. Examples: generate, plugin.
  • Sub-command = child command. Examples: generate posts, plugin status.
  • Option = command modifiers. Examples: generate -p post.
